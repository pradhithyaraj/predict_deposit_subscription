# Predict Term Deposit Subscription

A machine learning project that predicts whether a customer will subscribe to a bank term deposit based on marketing campaign data.

## Overview
This project focuses on analyzing bank marketing data, preparing features, and building classification models to understand and predict customer subscription behavior.

## Features
- Structured data preprocessing  
- Exploratory Data Analysis (EDA)  
- Multiple ML models for comparison  
- Model evaluation using standard metrics  

## Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Jupyter Notebook

## Project Structure
datasets/ # Data files
notebook/ # Analysis & model development
src/ # Preprocessing & model scripts


## Goal
Develop an efficient and interpretable model that supports targeted marketing and improves decision-making.
